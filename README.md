
# Linco Dev Blog

#### node 12.13.0 버전 이상 필요

```
nvm i 12.13.0
nvm use @12.13.0
```

#### 설치

```
npm i -g gatsby-cli
yarn install
```

####  실행

```
yarn start
```

#### 게시글 작성

<ol>
  <li>content/blog 경로의 폴더를 1개 복사</li>
  <li>index.md 내용을 수정</li>
  <li>썸네일 등록: 작성중인 마크다운이 있는 폴더에 이미지 파일 생성 후 " thumbnail: .... " 에 파일명 추가</li>
</ol>

#### 게시글 설정
> title : "게시글 제목"
> date: "시간 / 2015-05-01T22:12:03.284Z"
> description: "내용 요약"
> writter: "작성자"
> tag: "카테고리 명"
> thumbnail: "썸네일 salty_egg.jpg"
